# 🚗 Driving School - Web Application

![Go](https://img.shields.io/badge/Backend-Go-blue?logo=go)
![React](https://img.shields.io/badge/Frontend-React-61DAFB?logo=react)
![PostgreSQL](https://img.shields.io/badge/Database-PostgreSQL-4169E1?logo=postgresql)

Веб-приложение для автошколы с системой авторизации, записи студентов и управления расписанием.

---

## 🌟 Функционал

### ✅ Реализовано

**🔐 Авторизация студентов**
- 📝 Регистрация/вход (JWT)
- 👥 Ролевая модель (студент/админ)

**📋 Запись для поступающих**
- ✍️ Форма подачи заявки
- 🔍 Просмотр статуса заявки

### 🚧 В разработке

**🎓 Личный кабинет студента**
- 🗓️ Запись на практические занятия
- 📅 Просмотр расписания
- 📊 История пройденных уроков

**⚙️ Администрирование**
- 👨‍🏫 Управление инструкторами
- ✅ Подтверждение заявок
- ⚡ Генерация расписания
